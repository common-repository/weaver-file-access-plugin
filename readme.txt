Additional File Access support for the Weaver Theme.

== Description ==

This plugin for the Weaver theme allows Weaver 2.2 and later to work properly on hosts that require FTP access to create files. This plugin has no settings.

You must be using the Weaver WordPress Theme for this plugin to be useful.
